# Generic Shopping App

A simple Node.js shopping application built with Express and EJS.

## Requirements

- Node.js 16 or newer
- npm

## Install

1. Install dependencies:

```bash
npm install
```

2. Start the application:

```bash
npm start
```

3. Open http://localhost:3000 in your browser.

## Features

- Browse products
- Add items to cart
- Remove items from cart
